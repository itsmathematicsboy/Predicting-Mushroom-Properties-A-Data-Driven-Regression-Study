# Predicting-Mushroom-Properties-A-Data-Driven-Regression-Study

Project ini bertujuan untuk memprediksi kelas/kategori pada jamur berdasarkan bagian anatomi dari jamur. Model yang digunakan adalah dengan KNN Regressor.
Dataset terdiri dari 8 variabel fitur dan 1 variabel target. Dimana pada variabel tersebut antara lain, diameter mahkota jamur, warna, bentuk, tinggi, berat dan tumbuh dengan suhu tertentu.
untuk variabel targetnya, digunakan kolom kelas.

Library yg digunakan:
- Pandas
- Numpy
- Matplotlib
- Scikit-Learn

Metrik yang digunakan untuk mengukur performa model:
- Mean Absolute Error
- Mean Squared Error
- R2 Score

Capaian yang telah didapatkan:
- Mendapatkan MAE prediksi sebesar 0.016060
- Mendapatkan MSE prediksi sebesar 0.009271
- Mendapatkan R2 prediksi sebesar 0.962461
- Untuk Nilai K didapatkan paling optimal ada di nilai 5
